# Image Optimizer App

## Описание
Image Optimizer App - это приложение, разработанное с использованием Electron, Node.js и React, которое позволяет пользователям конвертировать изображения в формат WebP, оптимизировать их размер, перемещать все изображения в одну папку и обновлять пути к изображениям в CSS, JS и HTML файлах.

## Установка
1. Клонируйте репозиторий:
   ```
   git clone <URL_репозитория>
   ```
2. Перейдите в директорию проекта:
   ```
   cd image-optimizer-app
   ```
3. Установите зависимости:
   ```
   npm install
   ```

## Запуск
Для запуска приложения используйте следующую команду:
```
npm start
```

## Структура проекта
- `src/main`: Основной процесс приложения.
  - `main.ts`: Точка входа для основного процесса.
  - `ipc`: Обработчики IPC.
    - `handlers.ts`: Обработчики запросов от рендерера.
    - `channels.ts`: Определение каналов IPC.
  - `services`: Сервисы для обработки изображений.
    - `imageConverter.ts`: Конвертация изображений в формат WebP.
    - `imageOptimizer.ts`: Оптимизация изображений.
    - `fileOrganizer.ts`: Перемещение изображений в одну папку.
    - `pathUpdater.ts`: Обновление путей к изображениям.
  
- `src/renderer`: Рендерерный процесс приложения.
  - `index.tsx`: Точка входа для рендерерного процесса.
  - `App.tsx`: Основной компонент приложения.
  - `components`: Компоненты приложения.
    - `FileUploader.tsx`: Компонент для загрузки файлов.
    - `ProgressBar.tsx`: Компонент для отображения прогресса.
    - `ResultDisplay.tsx`: Компонент для отображения результатов.
  - `pages`: Страницы приложения.
    - `Home.tsx`: Компонент домашней страницы.
  - `styles`: Стили приложения.
    - `index.css`: Основные стили.

- `src/shared`: Общие типы и интерфейсы.
  - `types.ts`: Общие типы.

- `public`: Публичные файлы.
  - `index.html`: HTML-шаблон для рендерера.

- `package.json`: Конфигурация npm.
- `tsconfig.json`: Конфигурация TypeScript.
- `electron-builder.json`: Настройки для сборки приложения.
- `README.md`: Документация проекта.
